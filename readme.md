# dotfiles

Various dotfiles for configuring and personalising programs on Windows and Linux.

## Install

Install with:

```
# Powershell (Windows or Core)
. .\install.ps1 -User "Me", "You"
```

## Uninstall

Uninstall with:

```
# Powershell (Windows or Core)
. .\uninstall.ps1 -User "Me", "You"
```
